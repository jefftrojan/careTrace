# CareTrace OS — Healthcare Memory Infrastructure

AI-powered longitudinal healthcare intelligence platform. Transforms fragmented medical records into a continuously evolving patient memory graph.


## Features

- Longitudinal patient timeline reconstruction
- Multi-agent clinical reasoning (6 specialized agents)
- Patient intelligence relationship graph
- Physician mode + Patient mode dual views
- FHIR R4 interoperability output
- Clinical risk detection with explainability

## No secrets required

The backend API URL is baked into the build. No runtime configuration needed.
